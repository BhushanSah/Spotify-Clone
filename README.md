# Spotify Web Player UI Clone (HTML + CSS)

A front-end **Spotify Web Player** style UI built using **only HTML and CSS**.  
This project focuses on layout, styling, and responsiveness (no JavaScript / no real playback).

## Features
- Sidebar navigation (Home, Search, Your Library)
- Main content sections (Recently Played, Trending, Featured Charts)
- Sticky top navigation bar
- Bottom music player bar (static UI)
- Responsive layout with media queries

## Tech Stack
- **HTML5**
- **CSS3 (Flexbox + Media Queries)**
- **Google Fonts**: Montserrat
- **Font Awesome** icons
